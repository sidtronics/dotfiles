# Note
As this is non standard location for ratpoison config file ratpoison needs to started using "ratpoison -f ~/.config/ratpoison/ratpoisonrc"
So if you are using any login manager move the ratpoisonrc to ~/.ratpoisonrc. Else if you use startx add "exec ratpoison -f ~/.config/ratpoison/ratpoisonrc" in your .xinitrc. Check my .xinitrc for reference.
